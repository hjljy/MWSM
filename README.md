# MWSM
使用Vue + element UI 创建的一个简洁的个人网址导航 。可以直接下载使用，不涉及到后端服务，也可以自定义后端服务进行集成。
地址：http://www.lsbly.com

如有需要代码构建自己的网站导航，标明作者，修改仓库代码，自取即可。
